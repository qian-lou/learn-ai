# 缩放定律 / Scaling Laws

## 1. 背景（Background）
> Scaling Laws 揭示了模型性能与参数量、数据量、计算量的幂律关系。它指导了大模型的训练资源分配。

## 2-3. 知识点与内容
```
核心发现（Kaplan et al., 2020）：
L(N) ∝ N^{-0.076}  — 参数量 N 越大，Loss 越低
L(D) ∝ D^{-0.095}  — 数据量 D 越大，Loss 越低
L(C) ∝ C^{-0.050}  — 计算量 C 越大，Loss 越低

Chinchilla Laws（DeepMind, 2022）：
"模型参数量和训练数据量应该等比例增长"
70B 模型需要 1.4T token 训练数据（而非 300B token）
```

## 4-6. 推理/例题/习题
**练习：** 计算训练一个 7B 模型需要的最优数据量和计算成本。
