# 分布式训练 / Distributed Training

## 1. 背景（Background）
> 大模型参数量超过单卡显存，必须使用分布式训练。DDP/FSDP/DeepSpeed 是三种主流方案。

## 2-3. 知识点与内容
```
分布式策略对比：
数据并行(DDP): 每张卡一份模型，不同数据 → 梯度同步
模型并行: 模型拆分到多张卡
流水线并行: 模型按层切分，逐层执行
张量并行: 单层内部拆分到多卡

FSDP: 分片参数+梯度+优化器状态，显存效率最高
DeepSpeed ZeRO: 微软方案，ZeRO-1/2/3 递进式优化
```
