> **今日目标**：系统复盘 70 天所学，对照能力地图找差距，在「方向 A：Java 体系内」与「方向 B：纯 AI 工程」之间做出选择，并产出一份 90 天行动计划。
> **时长**：~2h ｜ **前置**：Day 1~69（五个里程碑项目 + 作品集 + 表达全部到位）
> **今日产出**：一份《能力自评 + 方向决策 + 90 天计划》文档（用下方清单/模板填实），外加一句你自己的"下一阶段定位"。

## 1. 为什么 & 是什么（概念 + Java 类比）

70 天到头，最容易犯的错是"学完就散"。复盘的价值不在回顾，而在**收敛**——把零散技能整理成一张能讲的能力地图，并据此**做一个决定**：往哪个方向深挖。方向选错不是学错知识，而是把有限精力摊薄在不形成合力的地方。今天就做这件最难也最值钱的事：**决策**。

给 Java 工程师的类比，都是你做过的"工程收尾"动作：

| 复盘动作 | Java/工程世界类比 | 要点 |
|---|---|---|
| 能力自评 | 技术雷达 / 能力矩阵自评 | 诚实打分，找出"会讲"与"真懂"的差 |
| 方向决策 | 技术选型评审（带取舍） | A/B 不是好坏，是**和你的存量优势匹配度** |
| 90 天计划 | 季度 OKR | 目标可量化、里程碑可验收 |
| 复盘文档 | 项目 retrospective / 复盘报告 | 沉淀成可回看的资产，而非脑内记忆 |

**核心心智：你的差异化是"Java 存量 × AI 增量"。** 市面上纯 AI 工程师很多，纯 Java 工程师更多，但**两者都能打通的人稀缺**。Day 65 那个混合系统就是这张王牌的具象化。两个方向的本质区别：方向 A 是"把 AI 能力装进 Java 体系、做企业级落地"，方向 B 是"投身 AI 工程前沿、深入推理/编排/评测底层"。选哪个，取决于你想让那张王牌**更稳**还是**更尖**。

**Step 2 — 方向决策（A vs B）**。先看对照表，再用下方打分模型量化：

| 维度 | 方向 A：Java 体系内 AI | 方向 B：纯 AI 工程 |
|---|---|---|
| 一句话 | 把 AI 装进企业 Java 体系做落地 | 投身 AI 工程前沿，深挖底层 |
| 典型岗位 | Java 架构师(带 AI) / 平台/中台 AI 落地 | AI/Agent 工程师 / LLM 平台 / 应用研究 |
| 技术重心 | Spring AI / LangChain4j / 企业集成 / 合规 | LangGraph 深水区 / eval 体系 / 推理优化 / 微调 |
| 复用存量 | ⭐⭐⭐ 极高（吃满 Java 老本） | ⭐⭐ 中（Java 经验做工程化加分） |
| 学习增量 | 中（在熟悉地基上扩展） | 高（要补 ML/系统/前沿） |
| 市场供给 | 稀缺（懂两边的人少） | 竞争更激烈但天花板更高 |

```text
方向打分模型（各维度 1~5，乘权重后求和，高者为主攻方向）
维度            权重   A得分   B得分
兴趣/热情        0.30    __      __
存量优势匹配     0.25    __      __
目标岗位机会     0.25    __      __
可投入学习强度   0.20    __      __
加权合计                ___     ___
结论：主攻方向 = ____，另一方向作为"不丢"的副线（每周留 1~2h 维持）。
```

> 提醒：**不必非黑即白**。务实路径常是"以 A 为主、保留 B 的深挖一两个点"（或反过来），用 Day 65 的混合系统作为两边都能讲的支点。决策的意义是**确定主次**，不是砍掉一边。

**Step 3 — 据方向定 90 天计划**（目标可量化、里程碑可验收）：

```text
90 天行动计划（按你选的主攻方向二选一填实）
目标(可量化)：例「拿到 N 个 AI 相关面试 / 上线一个真实流量项目 / 产出 M 篇技术文」
─ 第 1 个月：补强★清单 + 把作品集投出去（简历 + GitHub + 1 篇项目复盘文）
   里程碑：简历定稿、3 个项目 README/架构图齐全、投递 X 家
─ 第 2 个月：按方向深挖一个硬技能
   方向A：Spring AI/LangChain4j 做一个企业级集成 demo（带鉴权/可观测）
   方向B：LangGraph 复杂编排 + 自建 eval 体系 / 跑一次微调
   里程碑：新增 1 个深度项目或 1 套 eval 基准
─ 第 3 个月：面试冲刺 + 持续产出
   里程碑：完成 N 轮面试复盘、把高频问题答术打磨到能脱口而出
求职/深造下一步：__________（投递目标岗位 / 报哪门进阶 / 参与哪个开源）
```

**Step 4 — 写一句定位**：用一句话写下"下一阶段我是谁"。参考方向（不照抄）：

> "我是能把 LLM Agent 安全、可观测地落进企业 Java 体系的工程师——既懂模型的不确定性，也守得住生产的稳定性。"

## 3. 今日任务

1. **做能力自评**：六个 Phase 逐项打分并写"能讲清的证据"，标出★补强项。
2. **做方向决策**：填完 A/B 打分模型，明确**主攻方向 + 副线**，并写清理由。
