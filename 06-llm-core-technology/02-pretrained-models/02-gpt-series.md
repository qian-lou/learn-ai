# GPT 系列 / GPT Series (GPT-1/2/3/4)

## 1. 背景（Background）
> GPT 采用纯 Decoder 架构，通过自回归方式生成文本。从 GPT-1(117M) 到 GPT-4(估计 1.7T)，涌现了惊人的能力。

## 2-3. 知识点与内容
```
GPT 系列演进：
GPT-1 (2018): 117M 参数，证明预训练+微调的有效性
GPT-2 (2019): 1.5B 参数，zero-shot 能力涌现
GPT-3 (2020): 175B 参数，few-shot/in-context learning
GPT-4 (2023): 多模态，推理能力大幅提升

核心思想：自回归语言模型
P(text) = P(t1) × P(t2|t1) × P(t3|t1,t2) × ...
训练目标：最大化 Σ log P(t_i | t_1...t_{i-1})
```

## 4-6. 推理/例题/习题
**练习：** 用 GPT-2 做文本生成，对比不同采样策略的效果。
