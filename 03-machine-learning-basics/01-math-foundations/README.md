## 🔑 知识点详解

### 01 · 线性代数

- **核心概念**：一切数据都是向量与矩阵，模型的前向传播就是一连串线性变换（矩阵乘法）+ 非线性激活。
- **关键 API**：`A @ B`（矩阵乘）、`np.linalg.eig`（特征分解）、`np.linalg.svd`（奇异值分解）、`np.linalg.norm(v, ord=1/2)`（范数）。最该记的一式：余弦相似度 `dot(a,b) / (||a||·||b||)` —— Embedding 检索的度量基础。
- **易错点**：① 矩阵乘法不满足交换律，`A@B ≠ B@A`，写代码前先对齐维度 `[m,k]@[k,n]→[m,n]`；② `X^T X` 不满秩（存在共线特征）时不可直接 `inv`，要用伪逆 `np.linalg.pinv`。
- **Java 视角**：矩阵乘法类比嵌套 `for` 三重循环，但生产中绝不手写循环——NumPy 底层调 BLAS 向量化，正如你在 Java 里优先用成熟库而非裸循环。
- **前置**：阶段二 NumPy 的 ndarray 与广播机制。

### 02 · 概率与统计

- **核心概念**：分类模型输出的是概率分布；训练即最大化「数据在模型下出现的似然」，等价于最小化交叉熵。
- **关键公式**：Softmax `exp(zᵢ)/Σexp(zⱼ)` 把 logits 变概率；交叉熵 `H(p,q) = -Σ p·log q`，独热标签下退化为 `-log(p_正确类)`；贝叶斯 `P(A|B) = P(B|A)P(A)/P(B)`。
- **易错点**：① Softmax 要先减去最大值 `exp(z - max(z))` 防指数溢出；② `log(0)` 会得到 `-inf`/`NaN`，工程上一律加 `epsilon`（如 `1e-10`）或用 `np.clip`；③ 交叉熵 `H(p,q)` 不对称，`H(p,q) ≠ H(q,p)`。
- **Java 视角**：MLE 类比参数估计中「选一组参数让观测日志最可能发生」，和你按错误日志反推最可能的故障配置是同一种推断。
- **前置**：01 线性代数（Softmax 作用在 logits 向量上）。

### 03 · 微积分与优化

- **核心概念**：梯度指向函数上升最快的方向，所以「减去学习率×梯度」就朝极小值走；训练神经网络 = 反复做这一步。
- **关键公式**：梯度下降 `w ← w - lr·∂L/∂w`；链式法则 `∂y/∂x = ∂f/∂g · ∂g/∂h · ∂h/∂x`（反向传播就是它的自动化）；Adam 用一阶矩 `m` 和二阶矩 `v` 给每个参数自适应学习率。
- **易错点**：① 学习率过大发散、过小收敛慢，是最常调的超参；② 目标函数非凸（如神经网络损失）时梯度下降只保证局部最优，换初值/多次重启有用；③ 一个经典结论：Sigmoid + 二元交叉熵下，损失对输入 `z` 的梯度化简为极简的 `a - y`（预测值减真值），这是反向传播高效的原因之一。
- **Java 视角**：链式法则的逐层求导，类比调用栈自底向上回传返回值——`loss.backward()` 就像一次栈展开，把梯度沿计算图逐层传回每个参数。
- **前置**：01 线性代数（多维梯度是向量/矩阵）、02 概率（交叉熵作为被优化的损失）。

## 🎯 学习要点

- 线性代数是理解神经网络矩阵运算的基础：吃透「全连接层 = `Wx+b`」「Attention = `softmax(QKᵀ/√d)V` 三次矩阵乘」。
- 概率统计是理解损失函数和贝叶斯方法的核心：能说清「最小化交叉熵 = 最大似然」这条等价链。
- 梯度下降是几乎所有 ML/DL 优化算法的基石：手推一遍一维 `f(x)=(x+1)²` 的迭代收敛过程，建立肌肉记忆。
- 数值稳定性是工程必修课：Softmax 减 max、log 加 epsilon、梯度裁剪按范数——这些细节决定训练是否 NaN。
- 每个数学点都配一段 NumPy 代码亲手跑：看懂公式和跑通代码是两件事，先跑再理解形状变化。
- 不追求证明完备，追求「用得上的直觉」：能对着 Adam 更新式说出「为什么它比 SGD 稳」即可，暂不必推收敛性。
